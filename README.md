### Main Setting

**you can enable/disable mods, and also can choose time mode (Time Zone or Manual).**

![mains](https://cdn.modrinth.com/data/cached_images/a6195bb2d453eab4f6b059947eafebc495617e8d.jpeg)

### Manual Start Hour Setting

**You can set the time in Minecraft world. The top part is for hours and the bottom part is for minutes.**

![manual](https://cdn.modrinth.com/data/cached_images/c8b224a203735861c21841cd81a833e51fe4885b.jpeg)

### Country Time Zone

**Here there are several countries that you can choose from and the time will automatically adjust to that country.**

![kontritz](https://cdn.modrinth.com/data/cached_images/5b9189b9b206e20638fd19ee53766f66562dbfc9.jpeg)
